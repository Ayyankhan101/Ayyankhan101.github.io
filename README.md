## 🚀 Quick Start

### Prerequisites
- A GitHub account
- Basic knowledge of Git (optional, can use GitHub web interface)

### Deployment

#### Option 1: User Site (Recommended)
1. Create a new repository named `Ayyankhan101.github.io`
2. Upload `index.html` to the repository root
3. Commit changes
4. Visit `https://ayyankhan101.github.io` in 2-3 minutes

#### Option 2: Project Site
1. Upload `index.html` to any repository
2. Go to **Settings** → **Pages**
3. Select source: **Deploy from a branch**
4. Choose `main` branch and `/ (root)` folder
5. Save and wait for deployment

## 📝 Customization

### Update Personal Info
1. Edit the **Hero Section** for your name and tagline
2. Update the **About Section** with your bio
3. Modify **Skills Section** with your expertise
4. Add your projects in the **Projects Section**

### Add Your Projects
Copy this template and modify the details:
```html
<div class="project-card group relative rounded-2xl overflow-hidden bg-slate-800 border border-slate-700 reveal" data-category="data">
    <div class="p-6">
        <h3 class="text-xl font-bold mb-2">Project Name</h3>
        <p class="text-slate-400 mb-4 text-sm">Project description...</p>
        <a href="https://github.com/your-username/repo" target="_blank" class="text-blue-400 hover:text-blue-300">
            <i data-lucide="github" class="w-4 h-4"></i> View Code
        </a>
    </div>
</div>
```
